# Chong Gao · Academic Homepage

Personal academic website of **Chong Gao** (高冲), M.S. student at Sun Yat-sen University.

- **Live site**: <https://timchonggao.github.io/ChongGao.github.io/>
- **Style**: Jon Barron–style single-page homepage (Education / Experience / Publications / Skills / Honors)

## Tech Stack

- Static site generated by [Jekyll](https://jekyllrb.com/), based on the [academicpages](https://github.com/academicpages/academicpages.github.io) template (trimmed to a single page).
- Hosted on GitHub Pages from the `master` branch.

## Structure

```
ChongGao.github.io/
├── _pages/index.html   # the single-page homepage (main content)
├── _pages/404.md       # 404 page
├── _includes/          # head / seo / author-profile partials
├── _layouts/ _sass/    # theme layout & styles
├── images/             # profile photo, paper thumbnails (portfolio/), favicons
└── _config.yml         # site config (author, url/baseurl, seo)
```

## Local Preview

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```
